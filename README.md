# My_CRM_Project

This is a Modern CRM built with [Next.js](https://nextjs.org), [MongoDB](https://www.mongodb.com/), and [Tailwind CSS](https://tailwindcss.com/).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features Included
- Magic Lead AI Input (Powered by Google Gemini AI)
- Dark / Light themes
- Fully RTL UI (Hebrew)
- Responsive kanban boards, calendars, and customer data tables
- Robust user authentication and Role Based Access Control (RBAC)
